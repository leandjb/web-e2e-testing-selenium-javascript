# web-e2e-testing-selenium-javascript
This repository is designed for end-to-end web application testing using Selenium and JavaS

## Install:

Linux - OpenSUSE:
```
  sudo zypper install chromium 
```

Linux - Ubuntu:
```
  sudo apt-get [missing]
```

NPM:
```
  npm i selenium
```

## Excecute:
```
  node index.js
```
